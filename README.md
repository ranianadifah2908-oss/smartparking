# Smart Parking System

Smart Parking System using:
- ESP32
- MQTT Protocol
- Ultrasonic Sensor
- Flame Sensor
- Infrared Sensor
- Servo Motor
- Web Dashboard

## Features
- Real-time parking monitoring
- Flame detection alert
- Automatic gate control
- MQTT communication
- Responsive dashboard

## Technologies
- PHP
- JavaScript
- MQTT
- ESP32
- Chart.js
